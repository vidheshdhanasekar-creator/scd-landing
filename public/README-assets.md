# Asset Placement Guide

Place these image files in the `/public` folder:

| File name | Description |
|-----------|-------------|
| `aws-logo.png` | AWS logo (white on black) |
| `smvec-logo.png` | SMVEC college logo |
| `community-day-logo.png` | AWS Community Day Pondicherry skyline logo (use the dark/black version) |

The app has fallback text rendering if images are missing.
